The main essence of the project is to solve the “Don’t Get Kicked” Kaggle binary classification problem (predicting whether a used car is a “lemon”). The tasks cover: chronological split of data, preprocessing of categorical variables with OneHotEncoder and handling unseen categories, training baseline LogisticRegression, GaussianNB and KNN from sklearn with feature normalization, implementing Gini score, implementing custom versions of LogisticRegression (with SGD), KNN and NaiveBayes classifiers, reproducing sklearn results, creating non-linear features, selecting best features using logistic regression coefficients and L1 regularization, tuning hyperparameters. Implementing Precision, Recall, F1 and AUC PR metrics and recommending the most appropriate hard label metric for detecting lemon cars.
